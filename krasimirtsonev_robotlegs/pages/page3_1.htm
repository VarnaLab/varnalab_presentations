За и против използването на framework

  • насърчава използването на познати решения за познати проблеми
  • easier collaboration (пример: нов колега в екипа)
  • open source, community, a well-tested skeleton
  • less code to write
  • време за опознаване
  • трудно се излиза от рамките на framework-a
  • framework-ове се опитват да решат всички проблеми
  • black box code is hard to debug
